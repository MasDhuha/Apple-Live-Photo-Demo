Apple Live Photo Demo

This is a proof of concept demo using HTML5/CSS3 to create Apple Live Photo-equse effects. Making extensive use of the video tags support and functionality to create the effect. This demo is built on the Starter Bootstrap HTML, which utilizes frameworks (Bootstrap / Jquery / GoogleWebFonts) called in from a CDNs to save both time & weight.
